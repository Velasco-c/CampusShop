# 🛍️ Tienda de Ropa (HTML & CSS)

Este proyecto consiste en una tienda de ropa desarrollada únicamente con **HTML y CSS**, enfocada en la estructura visual, navegación y experiencia del usuario.

## 📌 Descripción General

La aplicación simula una tienda online donde los usuarios pueden explorar diferentes categorías de ropa, ver productos destacados, acceder a su perfil y navegar entre distintas secciones de forma intuitiva.

---

## 🧩 Estructura del Proyecto

El proyecto está organizado en múltiples páginas HTML y carpetas para estilos e imágenes:

* `/css` → Contiene los archivos de estilos.
* `/imagenes` → Imágenes de productos y recursos visuales.
* `/iconos` → Iconografía utilizada en la interfaz.

---

## 📄 Páginas Principales

### 🏠 Index (`index.html`)

Es la página principal de la tienda. Aquí se muestran:

* Secciones populares
* Navegación principal hacia otras páginas

---

### 👤 Perfil de Usuario (`user.html`)

Sección donde el usuario puede visualizar:

* Información personal
* Opciones relacionadas a su cuenta

---

### 🔐 Login (`login.html`)

Permite al usuario iniciar sesión en la tienda (interfaz visual).

---

### 🛒 Carrito (`carrito.html`)

Simula el carrito de compras donde se pueden ver los productos seleccionados.

---

### 💳 Pago (`payment.html`)

Página destinada al proceso de pago (simulado).

---

### 📦 Catálogo (`products.html`)

Muestra el catálogo general de productos disponibles en la tienda.

---

## 👕 Categorías de Ropa

### 👔 Camisas (`shirts.html`)

Sección dedicada a diferentes estilos de camisas.

### 👖 Pantalones (`pants.html`)

Muestra una colección de pantalones disponibles.

### 🧥 Abrigos (`coat.html`)

Incluye distintos tipos de abrigos.

---

## 🧭 Navegación

El sitio cuenta con un **header** común en todas las páginas que permite:

* Acceder rápidamente a cada sección
* Mantener una experiencia de usuario consistente
